# Obsidian Meeting Summary Template

Use this as the default shape unless the destination folder has a stronger local convention.

```markdown
---
type: meeting
origin: meeting-summary
state: working
people:
  - Person One
  - Person Two
tags:
  - meetings
created: YYYY-MM-DD
updated: YYYY-MM-DD
meeting_date: YYYY-MM-DD
---

# YYYY-MM-DD Meeting Title

## Summary

Brief work-focused summary of what mattered.

## Decisions

- Decision and rationale.

## Action Items

- [ ] Owner: action, due date if known.

## Open Questions

- Question or unresolved risk.

## Context

- Relevant project background, links, tickets, repos, or prior decisions.

## Source

- Provider: Granola | Fireflies | MeetJamie | Google Meet | Zoom | Slack | local file | pasted text | unknown
- Source reference: provider ID, local path, or URL when safe to store
- Source fingerprint: stable hash when available
- Raw artifact retained: no

## Safety Review

- Status: passed
- Removed categories:
  - Category only, with no sensitive details.
```
